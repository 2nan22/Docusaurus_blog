# Docusaurus 기술 블로그 만들기


### 환경설정

```
- Docusaurus 3.7 
- node.js 설치 (18.0 이상 버전)
- Git
```

### Git 설정
```
git init
```
```
git remote add origin ~
```
```
git config --global --user.name
```
```
git config --global --user.email
```
```
git pull origin main
```
```
cd ./Docusaurus_blog
```


### Node js 세팅
```
npm install
```
```
npm run start
```
